Javascript DOM (Document Object Model) Manipulation

When a web page is loaded, the browser creates a Document Object Model of the page.


The HTML DOM model is constructed as a tree of Objects:


With a programmable object model, JavaScript gets all the power it needs to create dynamic HTML:

  •     JavaScript can change all the HTML elements in the page
  •     JavaScript can change all the HTML attributes in the page
  •     JavaScript can change all the CSS styles in the page
  •     JavaScript can react to all the events in the page

Finding HTML Elements
Often, with JavaScript, you want to manipulate HTML elements.
To do so, you have to find the elements first. There are a couple of ways to do this:

  •     Finding HTML elements by id
  •     Finding HTML elements by tag name
  •     Finding HTML elements by class name

Finding HTML Elements by Id
Example
HTML

<p id="intro">Finding HTML elements by id</p>
Javascript
<script>
//-- Finding HTML elements by id
    x = document.getElementById("intro");
    document.write('<br> by ID => ' + x.innerHTML + '<br>');
</script>

Finding HTML elements by tagName
Example
HTML

<p id="P1" class="">Finding HTML elements by tagName</p>
Javascript
<script>
     //-- Finding HTML elements by tagName
    var y = document.getElementsByTagName("p");
    document.write('by tagName  => ' + y[1].innerHTML + '<br>');
</script>

Changing html content
Example
HTML

<p id="intro">Finding HTML elements by id</p>
Javascript
<script>
     //-- Changing html content
    document.getElementById("intro").innerHTML = "Text Changed by id";
</script>

Changing an HTML Attribute
Example
HTML

<img id="image" src="images/koohala.jpg" style="display: block;" />
Javascript
<script>
     //-- Changing an HTML Attribute
    document.getElementById("image").width = "100";
</script>

Changing HTML Style
Example
HTML

<p id="P1" class="">Finding HTML elements by tagName</p>
Javascript
<script>
    //-- Changing HTML Style
    document.getElementById("P1").style.color = "blue";
    document.getElementById("P1").style.fontWeight = "bold";
</script>

Onchange Event
The onchange event are often used in combination with validation of input fields.
Below is an example of how to use the onchange. The upperCase() function will be called when a user changes the content of an input field
Example
HTML

<input type="text" id="fname" onchange="myFunction()" style="display: block;">
Javascript
<script>
    function myFunction() {
        var x = document.getElementById("fname");
        x.value = x.value.toUpperCase();
    }
</script>

Onmouseover - OnmouseOut Event
The onmouseover and onmouseout events can be used to trigger a function when the user mouses over, or out of, an HTML element.
Example
HTML

<div onmouseover="mOver(this)" onmouseout="mOut(this)">Mouse Over Me</div>
Javascript
<script>
   function mOver(obj) {
        obj.innerHTML = "Yeah!"
    }

    function mOut(obj) {
        obj.innerHTML = "Mouse Over Me"
    }
</script>

Onmousedown - Onmouseup Event
The onmousedown, onmouseup, and onclick events are all parts of a mouse-click. First when a mouse-button is clicked, the onmousedown event is triggered, then, when the mouse-button is released, the onmouseup event is triggered, finally, when the mouse-click is completed, the onclick event is triggered.
Example
HTML

<div onmousedown="mDown(this)" onmouseup="mUp(this)">Click Me</div>
Javascript
<script>
   function mDown(obj) {
        obj.style.backgroundColor = "#1ec5e5";
        obj.innerHTML = "Release Me"
    }

    function mUp(obj) {
        obj.style.backgroundColor = "#D94A38";
        obj.innerHTML = "Yeah"
    }
</script>

Create  dynamic content
To add a new element to the HTML DOM, you must create the element (element node) first, and then append it to an existing element
Example
HTML

<div id="newp"></div>
<input type="button" value="Create" onclick="create()" />
<input type="button" value="Remove" onclick="removec()" />
Javascript
<script>
  function create() {
        var para = document.createElement("p");
        para.setAttribute('id', 'pd');
        var node = document.createTextNode("This is a js creation.");
        para.appendChild(node);

        var element = document.getElementById("newp");
        element.appendChild(para);
    }

   function removec() {
        var parent = document.getElementById("newp");
        var child = document.getElementById("pd");
        parent.removeChild(child);
    }
</script>
