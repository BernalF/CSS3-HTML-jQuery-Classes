LESSON 6 - BACKGROUNDS!

By Caleb Briancesco 09/10/13 at 04:41 pm

CSS

Background Color:
The background color sets the background color of any element, it will accept hexadecimal values, rgb values, rgba values (and a few more that we’ll not see at the moment).

Background-color: #ddeeff;

 

Background Image:
This attribute will set an image as the background of the element. In this case we can select the path of the image starting from the location of the css file we’re working on.
For example: If we have two folders, one for the css and one for the images and those two folders are in the same level, like in the next image  (image 01); we will need to ask the css to find the image file outside its content folder and look into another one, we can get out of our own container folder using this: “../”. By using ../ we are telling the css to look back out side our folder and we can still add more specifications so be can find the file such as the name of the next folder we would like to look at and the name and extension of the file we’re looking for.
For example: ../images/my_image.jpg

Background-image: url(../images/my_image.jpg);

 

Background Repeat:
In case we’re using an image as our background we can use many extra attributes to add position or size or many other kinds of ways to make it work, one of those values is “background repeat” and it works by using the same image we set as background but using it multiple times to cover all the space of the element we’re adding this attribute.

We can also use the background repeat by repeating the background in three different ways:
• Repeat
• Repeat-x
• Repeat-y

Repeat is the default value and it means that the background will be multiply by the x and y values at the same time.

Repeat-x and repeat-y will multiply the background image by each specific value.

If we don’t want the background to be repeated we can just set the value of background-repeat to no-repeat.

background-repeat:no-repeat;

 

Background position:
Wen we have a background we can set its vertical and horizontal position with this attribute; it also works with the x and y values respectively and we can set four different preset values and numeric values as well.

Background position requires 2 values to be applied, those values must be specified if we are working with numeric values but the browser can infer one of them if we use the preset values:
• Top
• Left
• Bottom
• Right

If we set the value to left it will apply also the center value so it will actually apply left and center,  if we apply center, if will also apply the center as second value so the browser will apply center and center.

We can set all this attributes separately or all of them in a single line separately with a single space by using the attribute background like this:

background:url(../images/image.jpg) no-repeat -250px -30px #13F7FF;

We can also apply multiple backgrounds at the same time.
