Lesson 03 - HTML Introduction

By Manuel Azofeifa 08/14/13 at 10:30 am

HTML, HTML5, CSS, CSS3, Web Development, Web Design

Ordered Lists

    By default this list starts with a number or an ordered sequence of characters   

<ol>
<li>Coffee</li>
<li>Milk</li>
</ol>

     You could take advantage of the attribute named: Type to specified the sequence: Type = (1, A, a, I, i)

1.  Coffee
2.  Milk

 

Unordered Lists

    This list starts with a disc bullet

<ul>
<li>Coffee</li>
<li>Milk</li>
</ul>

     You can change the bullet by setting a different value to the Style Attribute named: list-style-type (disc, circle, square)

•  Coffee
•  Milk

 

Description Lists

    A description list is a list of terms/names, with a description of each term/name

<dl>
<dt>Coffee</dt>
<dd>- black hot drink</dd>
<dt>Milk</dt>
<dd>- white cold drink</dd>
</dl>

    For example

Coffee

    - black hot drink

Milk

    - white cold drink

Block Elements

    Block level elements normally start (and end) with a new line when displayed in a browser

    Examples: <h1>, <p>, <ul>, <table>, <div>

Inline Elements

    Inline elements are normally displayed without starting a new line.

    Examples: <b>, <td>, <a>, <img>.

Layout

    In HTML 4 the Element DIV is the most popular piece to create layouts

    Lets use this CSS file to create a simple Layout by using DIVS  

.screen {
    width:1000px;
    background-color:#0094ff;
    height:800px;
}
.header {
    background-color:#00ff90;
    height:50px;
}
.body {
    height:700px;
    background-color:#b6ff00;
}
.content {
    background-color:#ff6a00;
    height:700px;
    width:800px;
    float:left;
}
.aside {
    background-color:#b200ff;
    height:700px;
    width:200px;
    float:left;
}
.footer {
    background-color:#4800ff;
    height:50px;
}

    And here you can see the HTML file

<html xmlns="http://www.w3.org/1999/xhtml">

        <div class="header">

            </div>
            <div class="content">
            </div>
        </div>
        <div class="footer">
            This is a footer
        </div>
    </div>
</body>
</html>

FORMS

    Let's start talking about interactive pages using <forms>. If you want to receive information from the user, you must consider using FORMS

<form>
    .
    input elements
    .
</form>

    The Form tag receives two main attributes: ACTION which represents the URL where this data is going to be send and the METHOD which specifies the transport mechanism used to send the information

    Methods:

        GET: It sends the information of the form through the URL as parameters

        POST: It sends the information within the body of the message

   This ACTION is triggered through the SUBMIT button

    These are some of the controls you could use when building a interactive form
