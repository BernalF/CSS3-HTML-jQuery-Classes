Iframes

 

As software developer, sometimes is required to embebed content from another sources, ie: as requirement you need to load content from specific web page in order to display important information, or simply display a document; for thus, you can go over using an iFrame element. The iframe element give us powerful control of his actions, so, you can define which types of documents you would like to display, or if you allow scripts execution, or load contents from another sources than the original.

 

Advantages

 

As advantages of iFrames using, we can mention:

  • Provides a measure of separation between the application and the content to load.
  • Won’t have access to your page’s DOM, or data you’ve stored locally.
  • Won’t be able to draw to arbitrary positions on the page.
  • It’s limited in scope to the frame’s outline.
  • Restrictions and policies could be applied.

 

How to declare it?

 

  • <iframe name="myFrame"></iframe>     (Here we're declaring an iframe which name is "MyFrame").
  •  
  • <iframe name="myFrame" sandbox="allow-same-origin allow-scripts allow-popups allow-forms“ src=""></iframe> (Now, we're declaring an iframe with the same name that the previous one, but, using the sandbox attribute we're delimiting the functionality of the iframe).

 

Loading content into iFrame

 

If you type the following code in your html web page:

 

             <a  href="http://www.advanceme.com/" target="myFrame">View Web Site 1</a>

 

You will be able to load the content of advanceme.com into the frame (MyFrame) that we declared previously.

Now, if you type:

 

               <a  href="Documents/public.txt" target="myFrame">View Content 2</a>

 

You will be able to load the content of the public.txt file into the the frame (MyFrame) that we declared previously.

Javascript Introduction - Class 01

 

What Is JavaScript?

  • Was designed to add interactivity to HTML pages

  • Is a scripting language (a scripting language is a lightweight programming language)

  • JavaScript code is usually embedded directly into HTML pages

  • JavaScript is an interpreted language (means that scripts execute without preliminary compilation)

Why Javascript?

  • Mobility

  • Cloud & Server-Side Development

  • Becoming The Default Enterprise Programming Language

  • Job Oportunity

 

The real world

  • SitePoint, an Australian book publisher focused on developers, compared data from job vacancies for 2012 and showed that JavaScript demand has increased faster than any other language. JavaScript doubles Ruby, Objective C and Android (Java), and over three times more than C++ and C#.

  • For June 2013 it showed that JavaScript was the fifth fastest growing language, year over year, after Objective C, PHP, Java and Python.

 

What can a JavaScript do?

  • JavaScript gives HTML designers a programming tool

  • JavaScript can put dynamic text into an HTML page

  • JavaScript can react to events

  • JavaScript can read and write HTML elements

  • JavaScript can be used to validate input data

  • JavaScript can be used to detect the visitor's browser

  • JavaScript can be used to create cookies

  • In resume, anything its possible with JavaScript

Types of data in Javascript
JavaScript has dynamic types. This means that the same variable can be used as different types
Examples

  •   var x;               // Now x is undefined
  •   var x = 5;           // Now x is a Number
  •   var x = "John";      // Now x is a String

 

Number
Support decimal, negative, exponential, etc
Examples

  •   isNaN(NaN);   //return true
  •   isNaN(10);   //return false
  •   isNaN(’10’);   // return false
  •   isNaN(‘cool’);  //return true

Note: IsNaN means "not a number" javascript native function.

 

String ("" & ')
String is a variable which stores a series of characters like "John Doe". A string can be any text inside quotes. You can use single or double quotes
Examples

  •   var x="He is called 'Johnny'";
  •   var x='He is called "Johnny"';
  •   var x="He is called \"Johnny"\ ";

 

Boolean
Booleans can only have two values: true or false.

Undefined and Null
Undefined is the value of a variable with no value.
Variables can be emptied by setting the value to null
Examples

  •   var temp = null;
  •   alert(null == undefined) //true
  •   alert(null === undefined) //false
  •   var temp;
  •   alert(temp);   //undefined
  •   alert(typeof temp === ‘undefined’) //true

Note: If you have declared but not initialized the variable, it is considered undefined:

JavaScript: Reacting to Events
