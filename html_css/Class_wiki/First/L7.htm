Lesson 07 - Pseudo Elements Laboratory

By Cristopher Castillo 11/01/13 at 01:33 pm

CSS, CSS3

Pseudo Elements


A pseudo-element does exactly what the word implies. It creates a new element and inserts it before or after the content of the element that you’ve targeted.
The word “pseudo” is a transliteration of a Greek word that basically means “lying, deceitful, false.” So, calling them pseudo-elements is appropriate, because they don’t actually change anything in the document. Rather, they insert ghost-like elements that are visible to the user and that are style-able in the CSS. Briancesco, C

In the following list you'll find a list of pseudo-elements

Selector

Example

Example description

:link

a:link

Selects all unvisited links

:visited

a:visited

Selects all visited links

:active

a:active

Selects the active link

:hover

a:hover

Selects links on mouse over

:focus

input:focus

Selects the input element which has focus

:first-letter

p:first-letter

Selects the first letter of every <p> element

:first-line

p:first-line

Selects the first line of every <p> element

:first-child

p:first-child

Selects every <p> elements that is the first child of its parent

:before

p:before

Insert content before every <p> element

:after

p:after

Insert content after every <p> element

:lang(language)

p:lang(it)

Selects every <p> element with a lang attribute value starting with "it"

 

Laboratory

 

Once is defined what a pseudo-element is, let's start to working on; please follow this steps:

 

  1. Create a new folder called "PseudoElements_Lab"
  2. Add a new folder called Css inside the folder created in the previous srep.
  3. Add a new web page in the PseudoElements_Lab folder root called "index.html"
  4. The content of index.html should look like:

<!doctype html>
<html>
<head>
    <meta charset="UTF-8">
    <title>Pseudo Elements Laboratory</title>
    </head>
<body>       
   
</body>
   
</html>

5. Add a new cascading style sheet file called defaults.css, please move it into the folder "Css" and then reference it  into the index.html web page. Should look like this:

<!doctype html>
<html>
<head>
    <meta charset="UTF-8">
    <title>Pseudo Elements Laboratory</title>
    <link rel="stylesheet" type="text/css" href="Css\reset.css" />
    </head>
<body>        
</body>   
</html>

 

Hover 

2- Please go to the stylesheet "defaults.css", and create a new class called MainContainer,as follows:

 

.MainContainer{

    width:1000px;

    height:600px;

    display:block;

}

 

3- Please go to the index.html web page, and then, inside the body tag please add a new <div> with the class "MainContainer":

    <div class="MainContainer">       

    </div>

 

4-Please go to defaults.css defaults and add a class called  Ulhover as follows:

 

.Ulhover

{

    background-color:#ffdb00;

    color:white;

}

 

.Ulhover li:hover {

    background:#00FF33;

    color:red;

}

 

With this, we are going to set the background color and text color of all elements <li> inside the class Ulhover

 

Please back to index.html  and insert inside the <div> with the Maincontainer class the following:

 

  <ul class="Ulhover">

            <li> This is a li number 1 </li>

            <li> This is a li number 2</li>

            <li> This is a li number 3</li>

            <li> This is a li number 4</li>

        </ul>

 

 Before

 

  1. Please go to defaults.css and type the following:

 

.Ulbefore{

    background-color:#ffdb00;

    color:white;

}

.Ulbefore li {

    background:#00c5c5;

    color:#000;

    padding:10px;

}

    2. Now, please go to index.html and add the following lines inside the <div> that has the MainContainer class:       

   

         <ul class="Ulbefore">

             <li> Cristopher Castillo</li>

             <li> Juan Perez</li>

             <li> Ana Rojas</li>

             <li> Felipe Vargas</li>

        </ul>   

 

   3. So, in the defaults.css file please type:

 

.Ulbefore li:before {

   content:"Your name is: ";

   font-weight:bold;

}

 

With this, we're going to search all elements <li> inside the class Ulbefore and we're going to insert the content :"Your name is"

    4. Now, lets do it a pseudo-elements combination, please go to the defaults.css and type the following:

.Ulbefore li:hover:before {

   background:red;  

}

 

After

Please go to defaults.css and add the following class

.Ulafter

{

    background-color:#ffdb00;

    color:white;

}

Let's add another  <ul> which class is " Ulafter" y we're going to add as <li> attributtes as you want, like this:

 

<ul class="Ulafter">

            <li championship="29"> Deportivo Saprissa</li>

            <li championship="28"> Alajuelense</li>

            <li championship="23"> Herediano</li>

            <li championship   ="6"> La Libertad</li>

        </ul>

Now, please go to the defaults.css file and type the following:

.Ulafter li {

    background:#FF00F3;

    color:#000;

    padding:10px;

}

 

.Ulafter li:after {

   content: " championships: " attr(championship);

}

That's going to let us to change the backgroud color of each <li> element that is inside the class “Ulafter” and finally we're goint to take the “championship” attribute and append a value.

 

First Line

 

Please go to defaults.css and type::

 

.UlfirstLine

{

    background-color:white;

    color:black;

}

.UlfirstLine li p

{

    background-color:blue;

    color:white;

    font-size: 25px;

}

 

Now please go to the index.html and add a new <ul> which class is "UlfirstLine" and we're going to add a new <p> attribute as that:

 

<ul class="UlfirstLine">

            <li>             

                <p>

                     Theodore Roosevelt

                    26.º presidente de los Estados Unidos de América (Nueva York, 1858 - Sagamore Hill, Nueva York, 1919). Historiador educado en       Harvard y procedente de una familia acomodada, fue un polifacético y un hombre de acción

                </p>

            </li>

        </ul>

****As you remember, the only way to add content to a <ul> attributte is adding<li> attributes inside it.

 

Now, let's back to the defaults.css and add the following:

 

.UlfirstLine li p:first-line

{

    background-color:yellow;

    color:black;

    font-size: 80px;

}

 

With this, we're going to search all paragraphs \( and set the background color of first line with yellow) inside all <li> that are contained in the <ul> that has the class “UlfirstLine” .

 

Pseudo-Elemento First Letter

 

Please go to the defaults.css and add the following:

 

.Ulfirstletter

{

    background-color:white;

    color:blue;

}

.Ulfirstletter li p

{

    background-color:white;

        color:blue;

         font-size: 25px;

}

 

Now, let's back to the index.html and add a new <ul> with "Ulfirstletter" class and please add a new <p>:

 

<ul class="Ulfirstletter">

            <li>

            <p>

                Every afternoon, as they were coming from school, the children used to go and play in the Giant's garden.

 

     It was a large lovely garden, with soft green grass. Here and there over the grass stood beautiful flowers like stars, and there were twelve peach-trees that in the spring-time broke out into delicate blossoms of pink and pearl, and in the autumn bore rich fruit. The birds sat on the trees and sang so sweetly that the children used to stop their games in order to listen to them. 'How happy we are here!' they cried to each other.

            </p>

                </li>

        </ul>

 

And in the index.html file add the following:

 

.Ulfirstletter li p:first-letter

{

    background-color:yellow;

   color:#ff0000;

     font-size:xx-large;

}

 

With that, we're going to search inside the class "Ulfirstletter" all <li> contained in this class, and then all <p> elements and finally set the background color of the first letter of the first paragraph.

 

First and Last Child

 

Please go to the defaults.css and put the following:

 

.UlfirstChild li{

    background-color:black;

   }   

 

Now go to the index.html and please add a <ul> attribute that uses the class "UlfirstChild" and add <li> as many you need (please add almost three):


<ul class="UlfirstChild">

        <li > Value 1</li>

        <li > Value 2</li>

        <li > Value 3</li>

         <li > Value 4</li>

</ul>

 

So, in defaults.css please add:

 

.UlfirstChild li:first-child {

    background-color:gray;

    color:red;   

}

.UlfirstChild li:last-child {

    background-color:violet;

    color:red;  

}

 

With the previous two sentences, we're be able to set the background color of the last child and first child (<li> elements) inside the class "UlfirstChild".

 

Nth-child

With this kind of pseudo-elements we're going to be able to access attributtes by index or taking in account simple math formulas.

 

  1. Let's to the defaults.css file and add the following:

 

.UlnthChild :nth-child(odd)

{

background:red;

}

.UlnthChild:nth-child(even)

{

background:blue;

}

And in the index.html type:

 

<ul class="UlnthChild">

            <li > Odd</li>

            <li > even</li>

            <li > odd</li>

            <li > even</li>

        </ul>

 

With this we're going to set the background color of odd elements with red  and with blue when the element is even.

 

Now, we're going to set the background color of childs which index is divisible by 4 and 3 as well, please see:

 

First add the following in the defaults.css file:

 

.UlnthChildMath :nth-child(3n+0)

{

background:cyan;

}

.UlnthChildMath :nth-child(4n+0)

{

background:green;

}

And now go to index.html and add a <ul> that uses the class "UlnthChildMath" and add almost 12 <li>, as following:

 <ul class="UlnthChildMath">

            <li > 1</li>

            <li > 2</li>

            <li > 3</li>

            <li > 4</li>

            <li > 5</li>

            <li > 6</li>

            <li > 7</li>

            <li > 8</li>

            <li > 9</li>

           <li > 10</li>

<li > 11</li>

<li > 12</li>

        </ul>
